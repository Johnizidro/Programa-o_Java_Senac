Java Script

Sobre o Java: JavaScript é uma linguagem de script que roda no navegador (como Chrome, Firefox, Safari) e permite que os sites façam coisas dinâmicas, como responder a cliques, validar formulários, criar animações, carregar conteúdo sem precisar atualizar a página, entre outros. Por exemplo, quando você clica num botão e algo muda na página sem precisar recarregar, provavelmente é JavaScript funcionando. Também é usado para jogos online, a plicativos web, e até em servidores (com Node.js).

  • HTML cria a estrutura da página (os textos, imagens, etc.)
  • CSS define o estilo (cores, fontes, layout)
  • JavaScript controla o comportamento e a interatividade.
Comando Sintaxe Função
Var, const, let var = Maria; Const = Maria; Let = Maria Define uma variavel, sendo var uma varivel que pode ter seu valor alterado e utilizado em qualquer parte do comando, let so pode ser utilizado em um determinado bloco de comando, e const não tem seu valor alterado
Alert Alert("Bem-Vindo!") Imprime um dado na tela para visualização do usuário, utilizando uma caixa de aviso nativa do navegador
If/Else if() {} else() {} Estrutura condicional para controle de ações lógicas dos comandos
+,-,/,* resultado = var1 + var2 Comando aritiméticos para definição de contas dentro do bloco de comandos
GetElementById GetElementById("Container") Puxa determinado elemento de acordo com o seu id e aplica alguma mudança sobre ele
InnerText InnerText = "Novo Texto" Altera um texto de determinado elemento de acordo com a tag ou id que ele possui
function function BotaoClicl(){} Inicia uma função e os comando internos dela, como o click de um botão para fazer uma conta
Prompt var = prompt("Insira um valor") Inserção de valores no navegador, entrada de dados por parte do usuário
